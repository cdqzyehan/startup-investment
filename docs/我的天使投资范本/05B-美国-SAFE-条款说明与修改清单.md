## 修改清单

### 1. 基础文件

建议尽量采用 YC 官方 Post-Money SAFE 原文，仅就估值上限、投资金额和 Pro Rata Side Letter 作有限补充。

### 2. 不建议自行修改的事项

除非美国律师确认，不建议自行修改转换机制、Dissolution Event、Liquidity Event、Company Capitalization 定义、税务条款和证券法陈述。

### 3. 可通过 Side Letter 处理的事项

| 事项 | 建议处理 |
| --- | --- |
| Pro rata right | 使用 YC Pro Rata Side Letter 或律师定制函件 |
| 信息权 | 单独信息权函件，约定更新频率和保密义务 |
| MFN | 如必须设置，明确适用范围和触发证券 |
| 后续融资通知 | 通过简短信函约定通知义务 |

### 4. 签署前确认

公司应取得董事会批准、确认 SAFE 不违反现有融资文件、更新 SAFE 台账，并在下一轮融资模型中列示所有 SAFE 的稀释影响。
