# 使用说明

完整版文件： term sheet = 股权协议 + 增资协议 + 章程修正案

本说明只给自己使用，不需要发给创始人。正式发给创始人的文件是本目录下 `01`、`02`、`03`、`04A`、`04B`、`04C`、`06`、`07` 这些文件。

## 哪些文件可以直接发给创始人

| 文件 | 用途 | 是否可直接发给创始人 |
| --- | --- | --- |
| 01 保密协议 NDA | 资料披露前签署 | 是 |
| 02 投资条款清单 Term Sheet | 锁定投资核心条款 | 是 |
| 03 尽职调查清单 | 要求公司提供资料 | 是 |
| 04A 中国境内增资协议 | 人民币增资主协议 | 是 |
| 04B 中国境内股东协议 | 投后治理和股东权利 | 是 |
| 04C 章程修正案与股东会决议 | 公司内部授权和登记文件 | 是 |
| 06 交割清单 | 付款和登记核对 | 是 |
| 07 投后信息权与重大事项清单 | 投后附件 | 是 |
| 05A/05B/05C 美国文件 | 境外律师起草指令 | 否，不建议直接签署 |

## 占位符处理

正式使用前，搜索 `[`，逐项替换所有占位符，例如：

| 占位符 | 替换为 |
| --- | --- |
| `[公司名称]` | 公司完整工商登记名称 |
| `[统一社会信用代码/注册号]` | 营业执照上的统一社会信用代码 |
| `[投资人名称]` | 投资人姓名或主体名称 |
| `[投资金额]` | 实际投资金额 |
| `[签署日期]` | 文件签署日期 |
| `[交割日期]` | 投资款支付和交割日期 |

## 打印和签署前检查

1. 确认公司名称、投资人名称、金额、估值、持股比例在全部文件中一致。
2. 删除不适用条款和空白附件。
3. 每个签署页前插入分页符。
4. 检查表格不要跨页断裂。
5. 保留文件末尾的“本文件以最终签署版本为准。”

## 不应出现在正式文件中的内容

正式发给创始人的文件中不应出现：

- 工作底稿说明。
- 起草说明。
- 参考来源。
- 给自己的操作提示。
- “是否聘请律师审阅”的模板提醒。

## 美国文件说明

`05A`、`05B`、`05C` 只作为境外律师起草指令，不作为中国境内人民币天使投资的直接签署文件。
