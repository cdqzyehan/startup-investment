## 正文条款

### 1. 定义

1.1 “保密信息”指披露方或其代表向接收方披露的非公开信息，包括商业计划、财务数据、技术资料、客户名单、合同、知识产权、融资条款、资料室文件、访谈记录以及接收方基于前述信息形成的分析材料。

1.2 下列信息不属于保密信息：已经公开的信息；接收方在披露前已合法持有的信息；第三方无保密义务合法提供的信息；接收方未使用保密信息独立开发的信息。

### 2. 使用目的

2.1 接收方仅可为评估对 `[公司名称]` 的潜在投资使用保密信息。

2.2 未经披露方书面同意，接收方不得将保密信息用于竞争、招揽客户、挖角员工、规避交易、为第三方谋利或其他与投资评估无关的目的。

### 3. 接触人员

| 可接触人员 | 要求 |
| --- | --- |
| 投资人合伙人/雇员 | 确有知悉必要并承担内部保密义务 |
| 律师、会计师、顾问 | 为本次交易提供专业服务并承担保密义务 |
| 关联基金或共同投资人 | 仅限经披露方书面同意或交易惯例允许的范围 |

3.1 接收方应对其接触人员违反本协议的行为承担责任。

### 4. 资料返还和销毁

披露方书面要求时，接收方应在 `[5]` 个工作日内返还或销毁保密信息及复制件。因法律、监管或内部合规需要保留的备份信息，仍继续适用本协议。

### 5. 保密期限

一般保密信息的保密期限为 `[3]` 年；构成商业秘密的信息在其不再构成商业秘密前持续保密。

### 6. 违约责任

违约方应赔偿守约方因此遭受的实际损失、合理律师费、调查费、保全费和执行费。双方确认，泄密可能造成难以用金钱弥补的损害，守约方有权申请禁令或行为保全。

### 7. 非投资承诺

本协议不构成任何投资、继续谈判、排他或交易完成义务。

### 8. 法律适用和争议解决

本协议适用中华人民共和国法律。争议提交 `[仲裁机构/有管辖权法院]` 解决。
