# 第一部分 非约束性条款

## 1. 交易结构

### 1.1 增资入股

公司拟向投资人定向增加注册资本。投资人拟以现金方式认购公司新增注册资本人民币 `[新增注册资本]` 元，其余投资款计入公司资本公积——资本溢价。

### 1.2 投资金额

投资人拟向公司投资人民币 `[投资金额]` 元。

其中：

```text
计入新增注册资本：人民币 [新增注册资本] 元
计入资本公积——资本溢价：人民币 [资本公积金额] 元
```

### 1.3 注册资本变更

本次投资完成后，公司注册资本由人民币 `[原注册资本]` 元增加至人民币 `[新注册资本]` 元。

### 1.4 投后股权比例

本次投资完成后，投资人持有公司 `[投资人持股比例]%` 股权。

本次投资完成后的股权结构以附件一《股权结构表 / Cap Table》为准。

### 1.5 新增注册资本计算方式

除各方另有书面约定外，本次新增注册资本可按以下公式计算：

```text
新增注册资本 = 本次投资完成前公司注册资本 × 投资人投后持股比例 ÷（1 - 投资人投后持股比例）
```

例如：

```text
公司原注册资本：人民币 100 万元
投资人投后持股比例：10%

新增注册资本 = 100 万 × 10% ÷ 90% = 11.1111 万元
投后注册资本 = 111.1111 万元
```

### 1.6 完全稀释口径

如本文件或正式交易文件约定按完全稀释口径计算，则“完全稀释口径”指假设公司已发行、预留或承诺发行的员工期权、限制性股权、可转股工具、代持权益、虚拟股权及其他类似权益均已完全行权、转换或落实后的股权口径。

### 1.7 员工激励池

如公司拟在本轮投资前设立或预留员工激励池，应在本轮投资交割前完成，并由创始人及原股东承担相应稀释，除非投资人书面同意由全体股东共同承担。

如公司拟在本轮投资完成后设立或扩大员工激励池，应经投资人事先书面同意，并按正式交易文件约定执行。

## 2. 投资款用途及监管

### 2.1 投资款用途

投资款仅应用于公司主营业务，包括但不限于：

1. 产品研发；
2. 产品标准化；
3. 设备采购；
4. 核心人员招聘；
5. 市场拓展；
6. 客户交付；
7. 服务器、软件、测试和实验费用；
8. 日常经营和运营支出。

### 2.2 禁止用途

未经投资人事先书面同意，投资款不得用于：

1. 股东分红或变相分红；
2. 偿还创始人、股东、员工或关联方个人债务；
3. 向创始人、股东、员工、关联方或其他第三方提供借款、担保、垫款或资金拆借；
4. 购买与公司主营业务无关的资产；
5. 支付明显不合理的关联交易费用；
6. 投资、收购或参与与公司主营业务无关的项目；
7. 其他与公司主营业务无关的支出。

### 2.3 资金使用报告

公司应每季度向投资人提供投资款使用明细、银行流水摘要、重大支出说明及下一季度资金使用计划。

如公司发生单笔超过人民币 `[金额]` 元，或连续十二个月内累计超过人民币 `[金额]` 元的重大支出，公司应提前向投资人书面说明支出目的、金额、付款对象和商业合理性。

## 3. 投资款支付安排

### 3.1 分期付款

投资人拟向公司投资人民币 `[总投资金额]` 元，分 `[二/三]` 期支付。

| 期数 | 金额 | 付款条件 |
| --- | --- | --- |
| 第一期 | 人民币 [金额] 元 | 正式交易文件签署；公司股东会批准本次增资；公司和创始人完成付款前尽调资料交付；公司确认收款账户 |
| 第二期 | 人民币 [金额] 元 | 公司完成工商变更/备案、章程更新、股东名册记载、出资证明书签发；并达到 [里程碑一] |
| 第三期 | 人民币 [金额] 元 | 公司达到 [里程碑二]；并向投资人提交资金使用报告、经营更新和相关证明材料 |

### 3.2 里程碑示例

具体里程碑由各方根据公司业务情况在正式交易文件中进一步确认，可包括但不限于：

1. 完成标准化产品版本；
2. 完成样机交付或客户验收；
3. 签署不少于人民币 `[金额]` 元的有效销售合同；
4. 实现不少于人民币 `[金额]` 元的回款；
5. 完成核心技术模块开发；
6. 完成关键专利、软著或知识产权申请；
7. 完成核心团队招聘；
8. 完成特定客户试点、部署或复购；
9. 完成投资人认可的财务、法务或知识产权整改事项。

### 3.3 后续付款义务

未达到相应里程碑或付款条件的，投资人无义务支付后续投资款。

已支付投资款对应的股权比例、股东权利及公司义务不因后续投资款未支付而当然失效，除非各方另有书面约定。

### 3.4 付款账户

投资款应支付至公司名下银行账户：

```text
账户名称：[公司名称]
开户银行：[开户银行]
银行账号：[银行账号]
```

未经投资人事先书面同意，投资款不得支付至创始人、股东、员工、关联方或任何第三方个人账户。

### 3.5 收款确认和入账

公司应在收到每期投资款后 `[3]` 个工作日内向投资人出具书面收款确认，并在 `[10]` 个工作日内向投资人提供相应财务入账凭证。

投资款应按正式交易文件约定分别计入：

```text
实收资本/注册资本
资本公积——资本溢价
```

投资款不得记为：

```text
借款
往来款
预收款
合作款
代收款
其他非股权投资性质款项
```

## 4. 投资人权利

### 4.1 基本股东权利

本次投资完成后，投资人依法享有公司股东权利，包括但不限于：

1. 分红权；
2. 表决权；
3. 知情权；
4. 剩余财产分配权；
5. 依法转让股权的权利；
6. 法律法规、公司章程和正式交易文件约定的其他权利。

### 4.2 信息权

公司应向投资人提供以下信息：

| 信息类型 | 提供频率 | 内容 |
| --- | --- | --- |
| 经营更新 | 每季度 | 产品进展、客户进展、收入情况、团队变化、重大风险 |
| 资金使用明细 | 每季度 | 投资款使用情况、重大支出、现金余额 |
| 财务报表或管理账 | 每年度/每季度 | 利润表、资产负债表、现金流情况或管理口径财务数据 |
| 年度预算 | 每年度 | 下一年度收入、成本、费用、融资和用款计划 |
| 重大事项通知 | 及时 | 诉讼、仲裁、行政处罚、重大债务、重大合同变化、核心人员离职等 |

### 4.3 查阅权

在不影响公司正常经营且遵守保密义务的前提下，投资人有权合理查阅公司章程、股东会决议、财务资料、重大合同、知识产权资料、股东名册及其他与投资人权益相关的文件。

### 4.4 优先认购权

公司新增注册资本、发行期权、可转股工具或其他类似权益时，投资人有权按照其届时持股比例优先认购，但员工激励、并购对价、经投资人事先书面同意的战略发行除外。

### 4.5 优先购买权

创始人或其他股东拟转让其持有的公司股权时，投资人在同等条件下享有优先购买权。

### 4.6 共同出售权

创始人或实际控制人拟向第三方转让其持有的公司股权时，投资人有权按照相同价格和条件，与创始人或实际控制人按比例共同向该第三方出售其持有的公司股权。

### 4.7 反稀释保护

如公司未来以低于本轮投资价格的价格进行融资，投资人享有广义加权平均反稀释保护。

以下情形不适用反稀释保护，除非正式交易文件另有约定：

1. 经投资人事先书面同意的员工激励；
2. 经投资人事先书面同意的战略合作或并购对价；
3. 公司股份制改造、资本公积转增、股权拆分、合并等不改变各股东经济权益比例的事项；
4. 经投资人事先书面同意的其他发行。

## 5. 重大事项

公司和创始人承诺，未经投资人事先书面同意，不得实施或同意实施以下事项：

| 事项 | 处理方式 |
| --- | --- |
| 修改公司章程中影响投资人权利的条款 | 需投资人事先书面同意 |
| 增加、减少注册资本，发行任何股权、期权、可转股权益或类似权益 | 需投资人事先书面同意 |
| 合并、分立、解散、清算、变更公司形式 | 需投资人事先书面同意 |
| 出售、转让、许可、质押或处分公司核心资产、核心知识产权或主营业务 | 需投资人事先书面同意 |
| 改变公司主营业务方向 | 需投资人事先书面同意 |
| 单笔或累计超过人民币 [金额] 元的对外借款、担保、资产抵押或重大债务 | 需投资人事先书面同意 |
| 单笔或累计超过人民币 [金额] 元的关联交易 | 需投资人事先书面同意 |
| 向创始人、股东、员工、关联方或第三方提供借款、担保、垫款或资金拆借 | 需投资人事先书面同意 |
| 分红、利润分配、资本公积转增注册资本 | 需投资人事先书面同意 |
| 任命或更换 CEO、财务负责人、核心技术负责人 | 需投资人事先书面同意 |
| 核心知识产权转让、质押、许可给第三方或由第三方持有 | 需投资人事先书面同意 |
| 超出年度预算人民币 [金额] 元或 [比例]% 的重大支出 | 需投资人事先书面同意 |
| 设立、投资、收购或控制其他主体 | 需投资人事先书面同意 |
| 公司控制权变化 | 需投资人事先书面同意 |

## 6. 创始人和公司承诺

### 6.1 全职服务

创始人承诺全职服务公司。未经投资人事先书面同意，创始人不得在其他经营性主体中担任全职职务，不得以任何方式实质参与与公司主营业务相竞争的业务。

### 6.2 竞业限制

创始人在持有公司股权期间，以及不再持有公司股权或离开公司后 `[期限]` 内，不得直接或间接从事、投资、参与、任职、顾问或协助任何与公司主营业务相竞争的业务，但法律法规另有规定或投资人书面同意的除外。

### 6.3 知识产权归属

与公司业务相关的全部知识产权、技术成果、软件代码、硬件设计、算法模型、控制系统、客户资源、商业秘密、专利申请权、商标、域名、产品资料及其他核心资产均应归属于公司。

如前述资产目前由创始人、员工、关联方或其他第三方持有，公司和创始人应在交割前或投资人认可的期限内完成转让、授权、确认或其他权属规范安排。

### 6.4 核心资产不得外移

未经投资人事先书面同意，公司和创始人不得将公司核心技术、核心知识产权、客户资源、产品资料、供应链资源或其他关键业务资产转移至创始人、关联方或其他第三方名下。

### 6.5 创始人股权限制

创始人所持公司股权在 `[4]` 年内原则上不得转让、质押或设置其他权利负担，除非经投资人事先书面同意。

创始人拟转让股权的，投资人享有优先购买权和共同出售权。

### 6.6 创始人离职或严重违约

如创始人主动离职、严重违反对公司或投资人的义务、从事竞争业务、侵占公司资产、转移核心资产、虚假披露重大信息或严重损害公司利益，公司及/或其他股东有权依据正式交易文件约定处理其未归属或受限股权。

具体处理方式由各方在正式交易文件中进一步约定。

### 6.7 陈述和保证

公司和创始人应在正式交易文件中作出充分陈述和保证，包括但不限于：

1. 公司依法设立并有效存续；
2. 公司股权结构真实、清晰、完整；
3. 不存在未披露的股权代持、期权、可转股安排或其他类似权益；
4. 公司提供的资料真实、准确、完整；
5. 不存在重大未披露债务、担保、抵押、质押或其他负债；
6. 不存在重大未披露诉讼、仲裁、行政处罚或潜在纠纷；
7. 公司依法纳税，不存在重大税务风险；
8. 公司核心知识产权权属清晰，不存在重大争议；
9. 公司重大合同真实、有效，不存在重大违约；
10. 公司不存在未披露的关联交易、利益输送或同业竞争；
11. 公司未发生重大不利变化。

## 7. 交割条件

投资人支付第一期投资款的前提条件包括：

1. 各方签署正式交易文件；
2. 公司股东会或全体股东批准本次增资；
3. 公司现有股东放弃或依法处理优先认购权及其他可能影响本次交易的权利；
4. 公司和创始人完成投资人合理要求的业务、财务、法务、知识产权尽职调查资料交付；
5. 公司和创始人作出的陈述和保证在重大方面真实、准确、完整；
6. 公司未发生重大不利变化；
7. 公司确认投资款收款账户；

## 9. 退出、清算和分配

### 9.1 退出事件

1. 公司清算、解散或破产；
2. 公司控制权变更；

### 9.2 优先分配

在法律法规允许范围内，发生退出事件时，投资人有权优先收回其已实际支付的投资本金。

如投资人按其届时持股比例可分配金额高于其已实际支付投资本金的，投资人有权选择按持股比例参与分配。

### 9.3 非对赌安排

除各方在正式交易文件中另有明确约定外，本文件不设置以公司或创始人业绩未达标为触发条件的强制回购、业绩补偿或对赌安排。
