# 宽松条款版本使用说明

> 本目录文件为创始人友好/快速签署版。所有模板均按“表格 + 编号条款 + 附件 + 签署页”的签署底稿格式编写，适合复制到 Word/WPS 后继续修订。

## 版本定位表

| 项目 | 条款口径 |
| --- | --- |
| 适用交易 | 适用于熟人投资、小额跟投、快速 SAFE 或少钱少股的人民币天使投资。 |
| 经济保护 | 如为定价股权融资，原则上不设置优先清算或仅设置 1 倍非参与型优先分配；如为 SAFE/可转债，按对应工具处理。 |
| 反稀释 | 不设置特别反稀释；如后续机构轮要求统一安排，投资人按同类小股东标准处理。 |
| 治理控制 | 不设置董事席位和日常经营否决权，仅保留出售、清算、主营业务停止等结构性事项通知。 |
| 信息权 | 半年度或季度简要经营更新；如公司向其他天使提供更频繁更新，应同等提供。 |
| 创始人约束 | 创始人确认诚信披露、知识产权归属和保密义务，不设置复杂成熟期或回购机制。 |
| 交割条件 | 签署正式文件、公司批准、付款和股权/证券记录更新后即完成交割。 |

## 使用步骤

1. 先阅读 `../导出说明.md`，确认是否使用 Pandoc、Word/WPS 或 MkDocs 导出。
2. 从 `02-投资条款清单-Term-Sheet.md` 锁定核心商业条款。
3. 中国境内交易使用 `04A`、`04B`、`04C` 作为正式协议组合。
4. 美国交易仅将 `05A`、`05B`、`05C` 交给美国律师作为起草指令，不建议自行签署。
5. 正式签署前删除所有起草说明，补齐附件、披露清单和签署页。

## 签署底稿提示

本目录文件不应未经律师审阅直接用于真实交易。若要形成正式签署版，应删除说明性段落，保留正文条款、附件和签署页，并确认所有 `[占位符]` 已替换。
