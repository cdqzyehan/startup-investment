## 正文条款要点

### 1. 自动转换

在合格融资发生时，未偿本金及累计利息应自动转换为该轮发行的同类优先股或影子优先股，转换价格按条款表确定。

### 2. 到期处理

到期未发生合格融资的，由律师根据公司现金情况和交易谈判结果设置延期、转换或偿还机制。

### 3. 违约事件

可包括不付款、破产、重大陈述不实、未经同意出售重大资产、发行优先证券、停止经营或重大违法。

### 4. 投资人权利

季度经营更新、年度财务报表或管理账、年度预算、重大事项即时通知。 美国法文件中应通过 side letter 或 note purchase agreement 明确。
