## 正文条款

### 1. 资料提供义务

公司和创始人应按本清单提供真实、准确、完整的文件。无法提供或不适用的，应在备注栏说明原因。

### 2. 披露清单

公司应单独准备披露清单，列明所有会影响投资人投资决定的重大事项，包括股权瑕疵、知识产权争议、重大债务、关联交易、诉讼、行政处罚、核心人员离职风险和未披露承诺。

### 3. 重点核查事项

3.1 股权是否清晰，是否存在代持、对赌、回购、质押、冻结、优先权或未披露股东协议。

3.2 公司核心知识产权是否由公司拥有或合法使用，创始人、员工、顾问和外包方是否完成权利转让。

3.3 公司是否存在未披露债务、担保、税务、劳动用工、数据合规或重大合同违约风险。

### 4. 整改事项

尽调发现的问题按以下方式处理：

| 类型 | 处理方式 |
| --- | --- |
| 交割前必须完成 | 影响股权、主体资格、核心 IP、重大债务或重大违法风险的事项 |
| 交割后整改 | 不影响投资目的但需完善的文件、登记或合规事项 |
| 披露接受 | 投资人已知悉并同意承担的风险 |

### 5. 公司确认

公司和创始人确认，未故意隐瞒任何重大事实。投资人有权基于尽调结果要求补充资料、调整条款、推迟交割或终止交易。
