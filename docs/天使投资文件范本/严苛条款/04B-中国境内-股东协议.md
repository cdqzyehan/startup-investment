## 正文条款

### 1. 协议目的

本协议用于补充公司章程和增资协议，约定投资人作为公司股东享有的特殊权利、公司治理机制、创始人义务及退出安排。章程与本协议不一致的，各方应在内部关系中按本协议履行，并配合修改章程或采取合法替代安排。

### 2. 股东权利表

| 权利 | 严苛版本安排 |
| --- | --- |
| 信息权 | 月度财务报表、月度经营数据、季度管理层报告、年度预算、年度审计或管理报表、重大事项即时通知。 |
| 重大事项 | 重大事项范围较宽，未经投资人或投资人委派董事书面同意不得实施。 |
| 优先认购权 | 公司新增注册资本或类似权益时，投资人按持股比例优先认购 |
| 优先购买权 | 创始人或其他股东转让股权时，公司和投资人享有优先购买权 |
| 共同出售权 | 创始人大额转让时，投资人有权按比例共同出售 |
| 反稀释 | 如公司后续融资的单位认购价格低于本轮价格，投资人享有完全棘轮或强广义加权平均反稀释保护；中国境内以股权补偿、创始人转让或其他合法方式实现。 |
| 退出分配 | 投资人享有投资本金 1 倍加已宣告未支付股息的优先分配；在取得优先分配后，投资人仍可按届时持股比例参与剩余分配，或由律师改为非参与型以降低后续融资阻力。 |

### 3. 公司治理

3.1 公司日常经营由管理团队负责。投资人根据本版本安排享有董事、观察员、重大事项同意权或通知权。

3.2 公司应保证所有股东会、董事会或执行董事文件真实、完整，并及时向投资人提供与其权利相关的会议通知、议案和决议。

### 4. 重大事项

未经投资人或主要投资人书面同意，公司不得实施下列事项，除非本版本明确仅要求通知：

| 事项 | 要求 |
| --- | --- |
| 修改章程或影响投资人权利 | 书面同意 |
| 新增注册资本、股权转让、期权计划重大调整 | 书面同意或遵守优先权 |
| 合并、分立、清算、出售公司或重大资产 | 书面同意 |
| 重大举债、担保、关联交易 | 超过人民币 [金额] 元需书面同意 |
| 主营业务重大改变 | 书面同意 |
| 核心知识产权转让或许可 | 书面同意 |

### 5. 创始人义务

创始人全职投入，股权设四年成熟期和一年 cliff，离职、欺诈、竞业或重大违约时未成熟部分可由公司或投资人回购。

5.1 创始人不得在未经投资人同意的情况下转让、质押或以其他方式处分其核心股权。

5.2 创始人应确保与公司主营业务相关的知识产权归属于公司或由公司合法、排他、持续使用。

### 6. 信息权

公司应按本版本信息权安排向投资人提供资料。投资人应对取得的非公开信息承担保密义务，不得干预公司日常经营。

### 7. 退出和转让

7.1 股东转让股权应遵守法律、章程和本协议约定。

7.2 发生公司出售、清算或控制权变更时，各方按本协议和章程约定分配交易对价。

7.3 拖售、共同出售、回购或赎回条款如涉及强制处分股权，应由律师确认可执行性后写入最终文本。

### 8. 特殊权利终止

公司完成合格 IPO、被上市公司并购、投资人不再持有公司股权或各方另行书面同意时，部分特殊权利可终止。

### 9. 违约责任和争议解决

违约方应赔偿守约方损失和合理维权费用。本协议适用中华人民共和国法律，争议提交 `[仲裁机构/有管辖权法院]` 解决。
