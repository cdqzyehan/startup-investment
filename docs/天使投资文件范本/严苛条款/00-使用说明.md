# 严苛条款版本使用说明

> 本目录文件为投资人强保护版。所有模板均按“表格 + 编号条款 + 附件 + 签署页”的签署底稿格式编写，适合复制到 Word/WPS 后继续修订。

## 版本定位表

| 项目 | 条款口径 |
| --- | --- |
| 适用交易 | 适用于投资金额较大、信息不透明、创始人约束和投后控制要求较高的交易。 |
| 经济保护 | 投资人享有投资本金 1 倍加已宣告未支付股息的优先分配；在取得优先分配后，投资人仍可按届时持股比例参与剩余分配，或由律师改为非参与型以降低后续融资阻力。 |
| 反稀释 | 如公司后续融资的单位认购价格低于本轮价格，投资人享有完全棘轮或强广义加权平均反稀释保护；中国境内以股权补偿、创始人转让或其他合法方式实现。 |
| 治理控制 | 重大事项范围较宽，未经投资人或投资人委派董事书面同意不得实施。 |
| 信息权 | 月度财务报表、月度经营数据、季度管理层报告、年度预算、年度审计或管理报表、重大事项即时通知。 |
| 创始人约束 | 创始人全职投入，股权设四年成熟期和一年 cliff，离职、欺诈、竞业或重大违约时未成熟部分可由公司或投资人回购。 |
| 交割条件 | 付款前须完成满意尽调、交易文件签署、内部批准、知识产权归属确认、创始人承诺文件、章程/工商或公司记录更新安排。 |

## 使用步骤

1. 先阅读 `../导出说明.md`，确认是否使用 Pandoc、Word/WPS 或 MkDocs 导出。
2. 从 `02-投资条款清单-Term-Sheet.md` 锁定核心商业条款。
3. 中国境内交易使用 `04A`、`04B`、`04C` 作为正式协议组合。
4. 美国交易仅将 `05A`、`05B`、`05C` 交给美国律师作为起草指令，不建议自行签署。
5. 正式签署前删除所有起草说明，补齐附件、披露清单和签署页。

## 签署底稿提示

本目录文件不应未经律师审阅直接用于真实交易。若要形成正式签署版，应删除说明性段落，保留正文条款、附件和签署页，并确认所有 `[占位符]` 已替换。
